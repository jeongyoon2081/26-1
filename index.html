1

면역 관용 (Immune Tolerance)

핵심: 자기항원을 인지하는 비정상 림프구를 제거하거나 불활성화시켜 자가면역을 방지하는 시스템. 중추(1차림프기관) vs 말초(2차림프기관) 면역관용으로 나뉨.
I. 자가 면역 관용 개요 ▶

정의

T 및 B림프구가 자신의 MHC만을 인식하면서 자기항원과는 반응하지 않도록 하는 기전.

  • T: 자신의 MHC만 인식, self antigen은 인식 X
  • B: 자신항원을 인식하지 X

의의

  • 알레르기 및 자가면역질환 예방·치료
  • 장기이식 거부반응 방지
  • 인위적 림프구 관용 유도 가능

면역관용의 2가지 단계

구분장소대상
중추(central) 면역관용1차 림프기관 (가슴샘, 골수)발생 초기 림프구
말초(peripheral) 면역관용2차 림프기관 (림프절, 비장, 점막림프계)중추에서 빠져나온 림프구
★ 면역관용은 미생물이 아니라 자신을 공격하는 비정상 림프구를 없애거나 불활성화시키는 것!
II. 중추 면역 관용 (Central Tolerance) ▶

1. T 림프구 (가슴샘에서)

관여하는 세포: 상피세포, 대식세포, 가지돌기세포

꼭 기억해야 할 2가지 ⭐⭐
  1. MHC restriction (MHC 제한): T림프구는 항원 peptide를 인지할 때 MHC도 동시에 인지해야 함
  2. 자기항원(self peptide)이 class I MHC(모든 유핵세포) 또는 class II MHC(APC)와 결합
왜 상피세포에 대부분의 self peptide가 존재하는가?
  1. 가슴샘 상피세포에 있는 AIRE (Autoimmune regulator protein)가 우리 몸의 거의 모든 단백질을 만들어냄
  2. 다른 곳의 자기항원이 혈관을 통해 가슴샘으로 들어옴

T cell 중추면역관용 과정

  1. Pre T (Double negative): 골수에서 CD4-, CD8- stem cell 생성
  2. Immature T (Double positive): 가슴샘 이동, CD4+, CD8+, CD3, TCR 발현
  3. 상피세포·대식세포·가지돌기세포와 접촉
  4. 인식 강도에 따라 운명 결정:
    • 인식 X → apoptosis
    • 강하게 인식 (MHC+self peptide) → apoptosis 또는 Treg 분화
    • 약하게 인식 (MHC만) → 생존 → 단일양성으로 분화
최종 결과: 자기 MHC는 인지하면서 self peptide는 인지하지 않는 TH, Tc만 살아남음. +) Treg도 생성!

2. B 림프구 (골수에서)

  • 자기항원과 강하게 반응하는 BCR → ① apoptosis 또는 ② Receptor editing (새로운 light chain 발현)
  • 약하게 반응하는 BCR → BCR 발현량 감소 → Anergy (무반응)

III. 말초 면역 관용 (Peripheral Tolerance) ▶

중추면역관용에 의해 제거되지 않은 자가면역반응 림프구를 제거.

B cell 말초관용

정상 B 림프구는 반드시 보조 T 림프구의 도움을 받아야 활성화. TH가 제거선택되었거나 Treg에 의해 보조자극(CD40L)을 받지 못하면:

  • ① 무반응 (anergy)
  • ② 세포고사 (apoptosis)
  • ③ 억제 수용체를 통한 BCR threshold 조절

T cell 말초관용 — 3가지 기전

① 무반응 (Anergy) — TH가 살아있되 무반응
• 정상: APC가 B7 충분히 발현 → TCR + CD28 (B7 인식) → 활성화
• 자기항원: B7 없거나, CTLA-4가 B7과 결합하여 제거
CTLA-4는 꼭 기억! (B7에 대한 친화력이 CD28보다 강함)
② 세포사 (Apoptosis)
• 정상: TCR(pro-apoptotic) + CD28(anti-apoptotic) → 활성화
• 자기항원: Fas-FasL 상호작용에 의한 apoptosis (반복자극 시) 또는 B7 없어 anti-apoptotic 단백질 X
③ Treg (조절/억제 T 림프구)에 의한 면역억제

특징: ⑴ CTLA-4 발현 ⑵ IL-2 소모 ⑶ IL-10, TGF-β 생성

억제 기전 ★★★:

  • 접촉의존: Treg의 CTLA-4가 APC의 B7과 결합 → B7 제거
  • IL-2 소모: 자기반응 TH가 증식하는데 필요한 IL-2를 사용
  • Cytokine 분비: IL-10 (DC, Mφ의 B7·class II·IL-12 발현 억제), TGF-β (TH1, TH2 분화 억제, Treg 분화 유도)
IV. 자가면역 & 유발인자 ▶

자가면역의 특성 ★

  • T, B 또는 두 세포 모두에서 면역관용 문제 발생
  • 유전적 요인 + 환경적 요인 (염증 유발: 감염 또는 조직손상)
  • 전신적 또는 장기특이적
⭐⭐ 면역 특혜 지역 (Immunological privileged sites)

"이건 꼭 기억" — Brain, Eye, Testis(고환), Uterus(자궁)

혈관내피세포의 tight junction, 면역억제 cytokine 생성, 림프구 불활성화·죽이는 분자 발현 → 면역반응 잘 안 일어남

1. 유전적 요인 ⋆⋆⋆

  • 한 가족·일란성 쌍둥이에서 발생빈도 높음
  • 특정 HLA 분자 발현 시 자가면역질환 확률↑ (relative risk)
  • AIRE, CTLA-4, Fas 유전자 변이와 연관

2. 환경적 요인 — 미생물이 자가면역을 유도하는 4가지 기전

  1. Microbe induces B7 in APC: 자기 peptide와 결합한 APC에서 B7 발현 유도 → 자기반응 TH 활성화
  2. 미생물 효소가 자기단백질 변형: 예) P. gingivalis의 peptidyl-arginine deiminase → rheumatoid arthritis
  3. Microbe has similar peptide to self (분자모방, molecular mimicry): 교차반응으로 자기반응 T 활성화 (예: CMV/HLA-DR, Poliovirus/ACh receptor)
  4. 면역특혜지역 손상: 외상·감염으로 격리항원 노출 → 자가면역 (예: 외상후 포도막염, 정관수술 후 고환염)

3. 기타 요인

  • 외상에 의한 격리조직(눈·고환) 손상
  • 과도한 자외선 노출 (핵단백질 유리)
  • 호르몬 (여자에서 발생빈도 높음 — Sjögren, SLE, Graves, MG, RA, MS 등)

면역학적 검사방법 (1) — 차정헌 교수님

출제 경향: 짤족 서술형 2-3문제 + 객관식 1-2문제. 검사 방법을 달달 외우기보다 기출 위주로 서술형 답안 작성 연습 추천. 각 실험에서 항원/항체 중 무엇을 detect하는지가 핵심!
1-4. 기초 검사법 ▶

1. Haemagglutination (적혈구 응집)

  • 목적: 혈청에서 적혈구(또는 적혈구 표면 코팅 항원)와 결합하는 항체의 유무·양 detect
  • 원리: negative → 항체-적혈구 결합 X, 바닥 중심에 진한 dot / positive → 결합, 희미한 큰 원
  • 96 well plate에서 가로방향 serial dilution → 어느 희석배수까지 positive인지로 항체량 측정

2. Complement fixation

목적: 보체를 이용해 혈청 내 항원에 대한 항체 존재 여부 검사

단계왼쪽 tube (Ag만)오른쪽 tube (Ag + test serum)
1항체 X → immune complex 없음항체 있음 → immune complex 형성
2Complement freeComplement → complex에 결합
3 (RBC 첨가)RBC lysis 일어남lysis 없음 (항체 있음)
결과 분석: Lysis 일어나면 항체 X / Lysis 없으면 항체 있음 (반대!)

3. Lymphocyte stimulation test

분리 방법 (꼭 알기): Density-gradient separation on Ficoll Isopaque — 밀도 차 이용한 원심분리. lymphocyte < ficoll < RBC·granulocyte

  • 목적: 특정 항원이 림프구를 stimulation 시키는지 검사
  • 방법: 분리한 lymphocyte + antigen + culture medium + [3H] thymidine 배양 → radioactivity 측정
  • 해석: 항원에 자극 → 림프구 proliferation → DNA 합성 시 ³H thymidine 흡수 → radioactivity ↑
★ thymidine을 쓰는 이유: 세포가 분열하면서 DNA를 복제할 때 thymidine을 흡수 → DNA 합성·증식 정도(=림프구 활성화) 측정 가능 (자주 출제!)

4. Cytotoxicity assay by chromium release

  • 목적: effector cell (NK, Mφ)의 cytotoxicity 측정
  • 방법: ⁵¹Cr로 target cell label → effector cell 첨가 → target cell이 죽으면 ⁵¹Cr 방출 → radioactivity 측정

6. Polyclonal Ab Purification by affinity chromatography

특정 항원에 대한 polyclonal Ab 정제. Antigen-bead column에 serum loading → wash → elution (pH 변화)

7. Immunoprecipitation

Antibody-bead complex로 특정 단백질(항원) 정제. Co-IP는 결합 단백질도 함께 detect 가능.

8. Agglutination tests for soluble antigens

Latex particle에 항체 코팅 → multivalent 항원과 결합 시 응집 → 가시화. Co-agglutination: Staphylococcus protein A가 항체 Fc와 결합하는 성질 이용 (예: S. pneumoniae 진단).

9-11. ELISA & Western Blot ▶

9. ELISA (Enzyme-Linked Immunosorbent Assay)

목적: 혈청 내 특정 항원에 대한 항체의 양 측정 → plate에 항원 코팅!

HIV ELISA 원리:

  • HIV는 잠복기 존재 → serum에 항원 X, 그러나 항체는 계속 존재
  • HIV 항원 코팅 plate → 환자 혈청 → washing → 2차 항체(anti-human Ig + enzyme) → washing → substrate → 발색

ELISA 단계: ① 항원 코팅 → ② 환자 혈청 → ③ Wash → ④ 2차 항체 → ⑤ Wash → ⑥ Substrate → ⑦ 발색 측정 (450nm OD)

12-13. ELISPOT & FACS ▶

12. ELISPOT (Enzyme-Linked Immunospot)

목적: 항체나 cytokine을 분비하는 개별 세포의 수 측정 (B cell이 항체 생성하는지, T cell이 cytokine 생성하는지)

분자실험 방법론 & 점막면역계 (MALT)

I. PCR & 분자기법 ▶

1. PCR (Polymerase Chain Reaction)

  • 두 종류 primer(forward, reverse)로 목적 DNA 영역 사이에 두고 증폭
  • 재료: Template DNA, Primer, dNTP, Taq polymerase, PCR buffer (Mg²⁺)
3단계 cycle (30-35회 반복):
  1. Denaturation (94°C, 15초~1분) — 2가닥 → 1가닥
  2. Annealing (42-55°C, 30초~1분) — Primer가 template에 결합
  3. Extension (72°C, 30초~10분) — Taq polymerase가 DNA 합성
Initial denaturation (94°C, 2-3분) + Final extension (72°C, 10분)

2. Southern Blotting

특정 DNA fragment detection. DNA를 restriction enzyme으로 절단 → 전기영동 → blotting → DNA probe로 hybridization → autoradiography/fluorescent

3. Sanger Sequencing

  • ddNTP (3'-OH 없음) 사용 → DNA polymerization 멈춤
  • 4종 ddNTP를 각각 다른 tube 또는 다른 색 형광으로 labeling
  • 전기영동 후 short→long 순서로 읽어 5'→3' 서열 결정

4. Cloning

  • Bacteria E.coli plasmid를 vector로 사용
  • 제한효소로 절단 → Sticky end (complementary, affinity↑) 또는 Blunt end
  • Sticky end가 ligation에 용이
  • 필수 조건: ① Replication origin ② Selective marker (예: 항생제 저항성 유전자)

5. RT-PCR (Reverse Transcription PCR)

핵심 Q: 어떻게 total RNA에서 mRNA만 선별적으로 RT?
A: Oligo-dT primer가 mRNA의 poly A tail에만 결합 → mRNA만 선별적으로 cDNA로 reverse transcription
  • Retrovirus의 reverse transcriptase 이용
  • mRNA → cDNA → 일반 PCR로 증폭

6-7. Real-time PCR (정량 PCR)

실시간으로 증폭되는 DNA 양 측정. CT값 (threshold 넘는 cycle 수)이 작을수록 초기 template 多.

방법특징
SYBR Greends DNA에 비특이적 결합. 싸지만 정확도 낮음. Single product detection only
TaqMan probeSpecific. 비쌈. Multiplex 가능 (여러 색 probe). Reporter + Quencher 구조

8. COVID-19 PCR 검사

+ssRNA virus. RNA 추출 → cDNA 합성 → PCR 증폭 → TaqMan probe → 형광 측정. Multiplex로 3개까지 검출.

9. DNA Microarray (서술형 출제) ▶
★ 2023, 2024 주관식 출제 — 매년 나옴!

DNA chip 만들기

  1. 20,500가지 gene에 specific primer 제작
  2. DNA isolation + PCR로 specific DNA product 생성
  3. DNA product를 유리에 dot으로 printing (Dot 하나 = PCR reaction 하나)

How arrays work

  1. 정상세포 mRNA → RT-PCR → cDNA를 Green으로 labeling
  2. 암세포 mRNA → RT-PCR → cDNA를 Red로 labeling
  3. DNA chip에 뿌려 hybridization (single strand cDNA → 상보적 부위에 결합)
  4. Scanner로 dot 색 읽기

II. 점막면역계 (MALT) 개요 ▶

MALT 종류

  • 호흡기: BALT, NALT
  • 소화기: GALT (소장 Peyer's patch ★★★, 맹장·대장 follicle)
  • Waldeyer's ring: Adenoid, palatine tonsil, lingual tonsil
  • Lacrimal gland, Salivary gland ★, Mammary gland
  • 생식기 (자궁 등)
⭐⭐⭐ 점막면역계 특징 (다른 면역계와 차이점)
  1. IgA 생성 — Isotype switching 후 주로 IgA (다른 곳은 IgG, IgM 등)
  2. Mucosa-oriented cell homing — 활성화된 림프구가 다시 점막면역계로 선택적 이동
하루 5g의 IgA 생성 (2/3는 sIgA), 5종류 항체 중 IgA 60-70%로 가장 多

점막면역계가 아닌 것 (시험에 자주)

  • 갑상선 (Thyroid), 부갑상선 (Parathyroid) — MALT 아님!
  • 골수, 가슴샘 (1차 림프기관)
III. GALT: Peyer's patch (가장 중요!) ▶

Peyer's patch 구조

  • 융모 없이 납작한 상피 하방, lymphocyte 모인 부분
  • Lumen 쪽 정상세균총 + Enterocyte 사이 M cell이 듬성듬성
  • Dendritic cell이 항원제시 (일부는 lumen까지 돌출)
  • 숫T (parafollicle), 숫B (follicle), 림프관 연결

M cell (Membranous / Microfold cell)

특징:
  • Enterocyte의 0.1% 미만, 섬모 없음
  • Antigen transport cell — pinocytosis로 항원을 Peyer's patch로 운반
  • ★★★ class II MHC 발현 X (항원제시세포 아님!) — 분해 X, 통로 역할만
⭐⭐⭐⭐⭐ Peyer's patch에서 T의존 항원에 대한 숫T와 숫B 활성화
  1. 항원이 M cell로 Peyer's patch로 이동
  2. Dendritic cell이 포식 → class II MHC + peptide로 숫T 활성화 → 작동 TH
  3. 또는 숫 B가 막결합 IgM/IgD로 항원 직접 인지
  4. 작동 TH와 숫 B 접촉 → 활성화 → 항체 분비
  5. IgA isotype switching ★★★:
    • 작동 TH의 CD40L + 점막면역계 DC/상피세포가 분비하는 TGF-β ★ → IgA 전환 유도
    • Retinoic acid ★가 작동T·B 표면에 α4β7 integrin과 chemokine receptor 발현 유도 → 점막면역계로 선택적 이동
  6. 작동 TH + plasmablast → 림프관 → Mesenteric LN → 다른 점막면역계로 이동
결론: 점막면역계에서 IgA를 주로 분비하고 점막면역계로만 이동하는 이유는 TGF-β와 retinoic acid가 많이 생성되기 때문!

Selective Mucosal Homing ★

  • 점막면역계 혈관내피세포에 α4β7 integrin과 결합하는 부착분자 (MAdCAM-1) 존재
  • +chemokine receptor와 결합하는 chemokine 존재
  • → 작동 TH·plasmablast가 MALT 상피 하방으로 쉽게 이동
IV. Dimeric IgA & sIgA ▶

구조: IgA dimer + Joining chain + Secretory Component

  • 30분 내 발생 → 즉시형 과민반응 (immediate hypersensitivity)
  • IgE 항체 + Mast cell에 의한 반응
  • 전 국민의 20%

  4. Sensitization (감작): IgE가 Mast cell의 FCεR에 결합
  5. 동일 항원 재노출: Allergen이 Mast cell 표면 IgE 2개 이상에 교차결합 → 활성화
  6. 탈과립: Vasoactive amine, lipid mediator, cytokine 유리
★ Mast cell 활성화 조건: 최소 2개 이상의 IgE에 allergen이 cross-link! 하나에 결합으로는 활성화 X

종양면역 (Tumor Immunology)

I. 암에 대한 면역반응 ★★ ▶

1. 암항원의 종류

  • Mutation이 생긴 self-protein
  • Oncogene protein, mutation된 tumor suppressor gene protein
  • 과발현/시기 부적절 발현 self protein
  • 감염 세포의 oncogene virus 항원

2. 4가지 면역세포

1) 세포독성 T 세포 (Tc) ★★

  1. APC가 암세포 자체 포식 → tumor Ag → Class I MHC 제시
  2. APC가 림프절로 이동
  3. 숫 Tc 활성화 (TCR + Class I MHC + tumor peptide + CD28-B7)
  4. Effector Tc 증식·이동
  5. 암세포의 Class I MHC + tumor peptide 인지
  6. Perforin, Granzyme → apoptosis

2) NK 세포 (Natural Killer) ★★★

특징:
  • 특이성 X, 활성화 과정 불필요 (T와 차이점)
  • IL-2로 활성 촉진 가능 (LAK cell)
  • Class I MHC 발현 X + 활성수용체 ligand 발현 세포 죽임
  • 항체로 도포된 세포 죽임 (ADCC)
  • Perforin, granzyme로 죽이는 기전은 Tc와 동일
  • IFN-γ 생성으로 M1 Mφ 활성화, TH1 분화 유도

3) 대식세포 (M1)

II. 종양 면역회피 기전 ▶

1) T의 종양세포 인식 어려움

  1. 종양항원 발현 ↓
  2. MHC 발현 소실·저하
  3. 항체에 의한 항원의 capping·endocytosis (항원 사라짐)
  4. Glycocalyx로 표면 덮음 → 면역세포 접근 어려움

2) T 세포 활성화 억제

  • ① 종양세포가 APC의 B7 발현 잘 유도 X
  • ② PD-L1 발현 → T의 PD-1과 결합 → 억제신호
  • ③ 면역억제 cytokine (IL-10: B7, class II 억제 / TGF-β: TH 분화 억제, Treg 유도)

3) MDSC (Myeloid-derived suppressor cell)

미성숙 neutrophil, monocyte, DC 포함 골수계 전구세포. IL-10, TGF-β 생성 → 면역 억제

4) M2 활성화 유도

  • TGF-β, IL-10 → T세포 억제
  • TGF-β, VEGF → 혈관 형성 (영양 공급, 성장)
  • MMP → 결체조직 분해 → 전이

5) 보체 활성화 어려움

종양세포 표면의 보체 조절단백 발현 → 보체 의존 세포 용해 회피

외우는 법 - "암세포의 5가지 도망 전략":
① 숨기 (항원·MHC↓) ② 가리기 (glycocalyx) ③ 속이기 (PD-L1) ④ 부하 부르기 (MDSC, M2) ⑤ 방패 (보체 조절)
III. 면역치료 전략 ▶

1) Dendritic cell vaccine

종양항원과 함께 배양한 숙주 DC 접종 (또는 종양항원 유전자 plasmid 접종)

2) Cytokine 투여 (IL-2)

3) 환자 T를 분리 → IL-2 처리 → 증식 → 재투여

4) CAR-T (Chimeric Antigen Receptor T) ★

구성: 항체 VL + VH (종양항원 인지) + TCR ζ chain signaling + CD28 signaling
장점: MHC가 없이도 종양항원 인지 가능 (MHC 발현 안 하는 암세포도 공격)

5) 항체 사용

  • 종양항원에 대한 항체 (보체·식작용·ADCC + chemotherapy drug 부착)
  • VEGF/EGF receptor 항체 (혈관·성장 차단)
  • Bispecific Ab (BiTE): Tumor Ag + T cell 활성수용체 연결
  • Anti-PD-1 / Anti-PD-L1 Ab: 면역회피 차단 (FDA 승인: melanoma 등)

종양 유발에서 면역의 역할

  • 만성염증 → 종양 유발 (H. pylori-위암, HBV/HCV-간암)
  • M1, Neutrophil ROS → 정상세포 DNA 손상 → oncogene
  • M2 → 혈관 형성·전이 촉진
  • → 암 예방을 위해 항염증제 사용

핵심: 이식실패의 가장 큰 이유 = 제공자 조직에 대한 받는자의 면역반응 (거부반응). HLA(인간 MHC)가 가장 중요!
주의: 제공자(donor)와 받는자(recipient)를 명확히 구분!

III. 거부반응 3종류 (초급성 / 급성 / 만성) ▶

1) 초급성 (Hyperacute) — 수 분~수 시간

  • 원인: 이식 전부터 있던 받는자 항체가 제공자 혈관내피의 항원(MHC, blood group)과 결합
  • 출처: 이전 수혈, 이식, 다임신, 장내세균총
  • 기전: 항체 → 보체 활성 + 염증 → 혈관내피 파괴 → 혈전 형성 → 괴사
  • 예방: Cross-matching (사전 항체 검사)

2) 급성 (Acute) — 수일~수주

  • 주로 T세포 매개. 직접·간접 인지로 활성
  • TH: cytokine → 염증 / Tc: apoptosis → parenchymal cell, 혈관 파괴
  • 조기이식 실패의 가장 중요한 원인
  • 예방: MHC 검사 (HLA typing) + 면역억제제

3) 만성 (Chronic) — 수개월~수년

  • 특징: 점차 좁아지는 혈관 + 섬유화
  • TH cytokine (IFN-γ) → 혈관 주변 평활근 증식 → 혈관 직경 좁아짐
  • 조직파괴 → fibroblast의 collagen 대체 (섬유화)
  • 예방 어려움 → 이식 실패의 주요 원인

4) 역거부반응 (Graft-versus-host)

제공자의 면역세포가 받는자의 조직을 공격. 조건: ① 제공자 조직에 성숙 T 多 ② 받는자 면역세포 결핍 (예: 백혈병 환자의 hematopoietic stem cell 이식)

IV. 예방 & 면역억제제 ★★★ ▶

사람마다 MHC가 다른 이유 3가지

  1. Class I, II MHC 유전자가 여러 개 (A, B, C / DP, DQ, DR)
  2. 각 유전자마다 여러 대립유전자 (Multiple Alleles)
  3. Codominant expression (양 부모 대립유전자 모두 발현)

HLA 적합성 검사 (HLA typing) — 급성·만성 예방 ★

3가지 방법:
  1. 보체의존 세포독성: B cell + HLA 항체 + 보체 + trypan blue (lysis 시 푸른색)
  2. PCR: Polymorphic region 증폭·시퀀싱
  3. MLR (Mixed Lymphocyte Reaction) ★★★: 한쪽에 mitomycin C 처리 (증식 억제) 후 혼합. 다른 쪽 림프구의 증식 정도로 MHC 유사성 판단. 증식이 작을수록 좋음!

Cross-matching ★★ — 초급성 예방!

받는자의 혈청에 제공자 HLA 및 기타 항원에 대한 항체가 있는지 확인

  • Complement-mediated cytotoxicity test
  • Flow cytometric assay
